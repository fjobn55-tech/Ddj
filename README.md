Lootbar Prototype (80% simulation)
Files included:
- React + TypeScript minimal app (Vite)
- Simple CSS to mimic Lootbar/v0 rounded dark UI
- Pages: Home, Games, FreeFire, Topup, Orders, Admin, QR
- Components: Header, GameCard, CategoryBar, OrderCard

How to run:
1) npm install
2) npm run dev

Notes:
- This is a frontend-only prototype (no backend API).
- Payment and QR are simulated placeholders.
